安装package contronl

package control是sublime text3的插件神器，安装各种插件就靠它了，是我们个性化配置我们自己的sublime text3的重要武器。命令行安装比较简单，使用control + ~打开sublime text3的控制台，直接粘贴

import urllib.request,os,hashlib; h = 'df21e130d211cfc94d9b0905775a7c0f' + '1e3d39e33b79698005270310898eea76'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

回车即可，很简单，可能有梯子速度会更快些。

安装sublime text3 nodejs插件

在sublime text3 下使用alt+shift+p调出查找工具窗口，输出package control install,这里有梯子能速度快些，输入nodejs回车即可，安装成功，不成功的话，手动去github上下载包。
接下来就是最为关键的一步，修改配置，我在这里跌倒太多次了，不同系统环境配置各不相同，但是都是大同小异的。
首先打开shubliem text3上的preference的Browser package,在文件夹nodejs里找到node.sublime-build文件，拖入到sublime text3中，修改两处：
（1）
"encoding": "utf8"修改encoding为编码格式为utf-8，防止编译文件出现乱码。
（2）

"linux":
    {
        "shell_cmd": "killall node; ~/.nvm/versions/node/v8.4.0/bin/node $file"
    },

修改shell_cmd,后面的node所在路径为我们安装的路径，如果使用nvm安装的话默认在~/.nvm/versions/node/v8.4.0/bin/node这个路径下，注意一定要加根目录~,否则是找不到路径的，（我在这卡了一晚上），保存，然后关闭。

然后再修改配置用户配置文件，在preference下的package control下选中nodejs,然后打开nodejs.sublime-settings文件，修改node路径和npm路径，把刚才修改node.sublime-build时所用的路径填写进去：

// e.g. "/usr/bin/node" or "C:\bin\node.exe"
"node_command": "~/.nvm/versions/node/v8.4.0/bin/node",
// Same for NPM command
"npm_command": "~/.nvm/versions/node/v8.4.0/bin/npm",

这样就大功告成了！

测试

编写文件server.js,使用nodejs语法，长这个样子：

var http = require("http")
var server = http.createServer(function(req,res){
    res.writeHead(200,{"Content-Type":"text/plain"})
    res.write("hello nodejs!")
    res.end()
})
server.listen(3000)
console.log("the server is running at port 3000")

然后按下control + B进行编译，编译结果会显示出来the server is running at port 3000,去浏览器输入127.0.0.1:3000就可以看到hello nodejs的字样了。
