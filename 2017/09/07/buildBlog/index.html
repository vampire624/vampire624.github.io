部署上线

下面我们需要部署上线，如果你是学生，其实可以尝试购买aliyun虚拟主机，但是需要一些linux以及服务器的各种配置，但是也是学习的过程了，对于想要马上看到自己博客能够部署上线的同学，那就尝试使用github，英文网站当然是，所以劳烦多学点英语哇，注册一个账号，新建一个repo， 注意！！！ 重点来了，repo的名字只能是你的github用户名，yourgithubName.github.io,github 能为我们提供page服务，非常轻松，国内的码云 也可以提供类似的功能，同学们可以自行百度啦，这样我们就有了一个可以存放我们代码的地方了，起到提供服务器的作用。

接下来，我们需要在开始的配置文件_config.yml中修改和添加最下面一行：

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
    type: git
    repo: https://github.com/yourgithubName/yourgithubName.github.io.git
    branch: master

修改后和上面一直，要把自己的github名字换掉！
